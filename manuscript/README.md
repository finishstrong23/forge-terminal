# The Hollow Hunt — revision workspace

Working files for the developmental edit of *The Hollow Hunt* (Book One of
the Hollow Series, R.M. Rime). The v1 "KDP ready" manuscript was assessed as
**not ready to publish** — see `editorial/verdict-2026-07-04.md`.

- `source/` — frozen v1: original .docx upload and extracted plain text.
  Never edited; kept as the dated pre-revision version for the rights file.
- `chapters/` — the living text, one file per chapter (plus front/back
  matter). All revision happens here, committed per pass.
- `editorial/` — the brief (`verdict-2026-07-04.md`), verified baseline
  numbers (`metrics-v1.md`), the phased plan (`revision-plan.md`), and
  `lint-prose.py`, which enforces the verbal-tic budgets.

Current status: v1 imported and verified; awaiting author decisions D1–D6 in
`editorial/revision-plan.md` before Pass 1 (beat sheet).
